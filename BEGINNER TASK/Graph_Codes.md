
# üìä Graph Types with Code Examples

This section includes commonly used plot types, each demonstrated with both **Matplotlib** and **Seaborn** examples.

---

## 1. üìà Line Plot  
**Use Case:** Visualizing trends over time.

### ‚û§ Matplotlib Example
'''
import matplotlib.pyplot as plt

x = [1, 2, 3, 4, 5]
y = [10, 12, 8, 15, 20]

plt.plot(x, y)
plt.title("Line Plot Example")
plt.xlabel("X Axis")
plt.ylabel("Y Axis")
plt.grid(True)
plt.show()
'''

### ‚û§ Seaborn Example
'''
import seaborn as sns
import matplotlib.pyplot as plt

data = sns.load_dataset("flights")
sns.lineplot(data=data, x="year", y="passengers")
plt.title("Passengers Over Years")
plt.show()
'''

---

## 2. üìä Bar Chart  
**Use Case:** Comparing values across categories.

### ‚û§ Matplotlib Example
'''
import matplotlib.pyplot as plt

categories = ["A", "B", "C", "D"]
values = [23, 45, 56, 78]

plt.bar(categories, values)
plt.title("Bar Chart Example")
plt.xlabel("Category")
plt.ylabel("Value")
plt.show()
'''

### ‚û§ Seaborn Example
'''
import seaborn as sns
import matplotlib.pyplot as plt

data = sns.load_dataset("titanic")
sns.countplot(data=data, x="class")
plt.title("Passenger Count by Class")
plt.show()
'''

---

## 3. üîµ Scatter Plot  
**Use Case:** Showing relationships between two variables.

### ‚û§ Matplotlib Example
'''
import matplotlib.pyplot as plt

x = [5, 7, 8, 7, 2, 17, 2, 9]
y = [99, 86, 87, 88, 100, 86, 103, 87]

plt.scatter(x, y)
plt.title("Scatter Plot Example")
plt.xlabel("X Value")
plt.ylabel("Y Value")
plt.show()
'''

### ‚û§ Seaborn Example
'''
import seaborn as sns
import matplotlib.pyplot as plt

data = sns.load_dataset("iris")
sns.scatterplot(data=data, x="sepal_length", y="sepal_width", hue="species")
plt.title("Sepal Size by Species")
plt.show()
'''

---

## 4. üìâ Histogram  
**Use Case:** Displaying data distribution.

### ‚û§ Matplotlib Example
'''
import matplotlib.pyplot as plt
import numpy as np

data = np.random.randn(1000)
plt.hist(data, bins=30, color='skyblue', edgecolor='black')
plt.title("Histogram Example")
plt.xlabel("Value")
plt.ylabel("Frequency")
plt.show()
'''

### ‚û§ Seaborn Example
'''
import seaborn as sns
import matplotlib.pyplot as plt

data = sns.load_dataset("diamonds")
sns.histplot(data=data, x="price", bins=30, kde=True)
plt.title("Price Distribution of Diamonds")
plt.show()
'''

---

## 5. ü•ß Pie Chart  
**Use Case:** Showing proportions (Matplotlib only).

### ‚û§ Matplotlib Example
'''
import matplotlib.pyplot as plt

labels = ['Apples', 'Bananas', 'Cherries', 'Dates']
sizes = [15, 30, 45, 10]

plt.pie(sizes, labels=labels, autopct='%1.1f%%', startangle=140)
plt.title("Fruit Distribution")
plt.axis('equal')
plt.show()
'''

---

## 6. üì¶ Box Plot  
**Use Case:** Summarizing distribution with min, max, median, and outliers.

### ‚û§ Matplotlib Example
'''
import matplotlib.pyplot as plt
import numpy as np

data = [np.random.normal(0, std, 100) for std in range(1, 4)]
plt.boxplot(data)
plt.title("Box Plot Example")
plt.xlabel("Dataset")
plt.ylabel("Value")
plt.show()
'''

### ‚û§ Seaborn Example
'''
import seaborn as sns
import matplotlib.pyplot as plt

data = sns.load_dataset("tips")
sns.boxplot(data=data, x="day", y="total_bill")
plt.title("Bill Distribution by Day")
plt.show()
'''
